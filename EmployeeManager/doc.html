Object Oriented Programming 2 - Final Project
Project Information
Project Title Employee Management System
Participant(s) YUANDONG YANG
Project Type Individual Project
Project Description This is a C# and PostgreSQL-based employee management system designed to help organizations efficiently manage their employees, departments, and wage structures. The system provides complete CRUD functionality, allowing users to manage employee records, department information, and wage grades. It also includes reporting capabilities, enabling organizations to analyze employee wage data and department distributions. Through the use of the Blazor framework, the system offers a modern, responsive web interface, enhancing the user experience.
Target Audience Human resources departments and management personnel in small to medium-sized businesses who need a simple yet efficient system to manage employee data, organizational structures, and wage information.
Project Motivation Businesses need to effectively manage their human resources data to improve organizational efficiency. Traditional paper records or simple spreadsheets struggle to meet the management needs of modern enterprises. This project aims to provide a comprehensive solution that simplifies employee data management and improves the accuracy and efficiency of human resources decision-making. Additionally, this project serves as a practical application of object-oriented programming and database design principles.
Design and Functionality
Main Features
  • Employee Management: Add, edit, delete employee information, including basic personal profiles and employment status management (termination and reinstatement)
  • Department Management: Create, update, and delete organizational departments
  • Wage Grade Management: Define and manage different salary grades, including base wages and bonus percentages
  • Reporting: Generate employee wage reports with detailed statistics and visualized charts
  • Data Validation: Verify the validity of input data, such as email formats and phone number formats
  • User-Friendly Interface: Intuitive navigation and modern design
User Interaction Users interact with the system through an intuitive web interface. The main dashboard provides an overview of the system's key functions and quick access links. Users can:
  • Add new employees, departments, or wage grades through forms
  • View existing records through tables, with search and filter capabilities
  • Modify existing records by clicking the edit button
  • Delete records through confirmation dialogs
  • Access the reports page to view statistics and charts
  • Manage employee employment status (termination or reinstatement)

The system employs responsive design, ensuring a good user experience across different devices. Error messages are clearly displayed, guiding users to take correct actions.

Technical Implementation

This project utilizes the following technologies:

  • C# programming language
  • Blazor frontend framework
  • PostgreSQL database
  • Dapper ORM framework
  • Object-oriented design principles
  • Layered architecture (data access layer, business logic layer, presentation layer)

The system has implemented all the features described in the proposal, including complete CRUD operations, report generation, and user-friendly interface design. The code structure is clear, follows object-oriented programming principles, and provides a solid foundation for further expansion and maintenance.
